# Xiaomi Push Fix Tool (Advanced)

Advanced ADB-based push notification fix & diagnostic tool for Xiaomi China ROM / HyperOS devices.

## Features
- FIX mode (apply all settings)
- VERIFY mode (diagnostics + report)
- COMBO mode (recommended)
- Multi-user (clone apps) support
- Logging system
- Works without root

## Usage
1. Connect phone via USB
2. Enable USB debugging
3. Run pushfix.bat
4. Choose mode

## Notes
- Re-run after reboot
- May increase battery usage
